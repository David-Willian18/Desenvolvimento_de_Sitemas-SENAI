JESUS' GEN: Cultura do Céu

Comunidade das Nações

Jesus' Generation ou Jesus' Gen (versão abreviada), representa a fusão dos ministérios da juventude da Comunidade das Nações, uma organização cristã com sede em Brasília, no Distrito Federal. Para a criação da sua identidade visual, criamos um sistema de elementos visuais que incorporam ícones, texturas, cores marcantes e elementos tipográficos, os quais intensificam os conceitos de unidade e conexão.
